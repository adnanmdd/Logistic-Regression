<h2>Logistic Regression: A Powerful Classification Algorithm</h2>

<p>
    <b>Logistic Regression</b> is a supervised machine learning algorithm used
    for <b>classification problems</b>. It predicts the probability of a data
    point belonging to a specific category using the <b>sigmoid function</b>.
    Unlike Linear Regression, Logistic Regression is mainly used for predicting
    discrete outcomes such as <i>Yes/No</i>, <i>True/False</i>, or <i>0/1</i>.
</p>

<h3>How It Works</h3>
<ul>
    <li>Collects and preprocesses input data</li>
    <li>Trains the model using labeled datasets</li>
    <li>Calculates probabilities using the sigmoid function</li>
    <li>Classifies outputs based on a threshold value</li>
</ul>
