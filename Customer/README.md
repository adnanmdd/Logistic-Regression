<h2>Customer Purchase Prediction Using Logistic Regression</h2>

<h3>Project Overview</h3>
<p>
This project focuses on predicting whether a customer will purchase a product or service based on
the parameter of <b>age</b> and <b>salary</b>. Since the outcome is binary (Purchase or No Purchase),
Logistic Regression is used as the machine learning algorithm to build the prediction model.
</p>

<h3>Applications</h3>
<ul>
    <li>Targeted marketing campaigns</li>
    <li>Customer segmentation</li>
    <li>Personalized product recommendations</li>
</ul>
